Project 1: Tic Tac Toe Game

Project 1
One of the most rewarding projects I've had the opportunity to undertake was the creation of a dynamic Tic Tac Toe webpage. This project showcased both front-end and back-end development prowess, offering users a fully functional gaming experience. On the front end, I crafted an interactive Tic Tac Toe game using a combination of HTML, CSS, and JavaScript. This allowed for seamless gameplay with features such as tracking the game's progress, determining winners and losers, and providing an option to restart for continuous enjoyment. However, the real challenge and highlight of the project came with the implementation of back-end functionality. Here, I leveraged powerful technologies like bcrypt for secure password encryption and MongoDB for database management. Each user's vital information, including usernames and securely encrypted passwords, was meticulously stored in the MongoDB database. This ensured robust security measures while maintaining user privacy and data integrity. In essence, this project not only honed my technical skills but also provided a comprehensive understanding of full-stack development, demonstrating my ability to seamlessly integrate front-end and back-end components to deliver a polished and engaging web experience.
